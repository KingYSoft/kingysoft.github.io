dotnet core 使用Apollo配置中心

发表于 2018-07-17 | 更新于 2018-07-20 | 分类于 .Net Core

Apollo（阿波罗）是携程框架部门研发的分布式配置中心，能够集中化管理应用不同环境、不同集群的配置，配置修改后能够实时推送到应用端，并且具备规范的权限、流程治理等特性，适用于微服务配置管理场景。本地快速部署请参见 Quick Start 。
